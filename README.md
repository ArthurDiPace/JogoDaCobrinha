# Jogo da Cobrinha em Python usando Pygame

Este é um jogo simples da cobrinha criado em Python usando a biblioteca Pygame.

## Pré-requisitos

Certifique-se de ter o Python e a biblioteca Pygame instalados na sua máquina antes de executar o jogo.

## Como executar

1. Clone o repositório para a sua máquina local.
2. Navegue até o diretório onde o repositório foi clonado.
3. Execute o seguinte comando no terminal para iniciar o jogo:

   ```shell
   python snake_game.py
   
## Controles
-Use as setas direcionais para controlar o movimento da cobrinha.

-Pressione a tecla Q para aumentar a pontuação em 10 pontos.

## Regras do jogo
-O objetivo do jogo é controlar a cobrinha e comer a comida que aparece na tela.

-Cada vez que a cobrinha come a comida, sua pontuação aumenta em 10 pontos.

-A cobrinha deve evitar colidir com as paredes da janela ou com o próprio corpo.

-O jogo termina quando a cobrinha colide com as paredes ou com seu próprio corpo.
